Описание задачи

<p>Напишите функцию <code>isPrime(n int) bool</code>, которая проверяет, является ли число простым. Простое число &mdash; это натуральное число больше 1, которое делится только на 1 и на себя.</p>

<p>Функция должна возвращать <code>true</code>, если число простое, и <code>false</code> в противном случае.</p>

<p>В программе вводится несколько чисел. Для каждого числа вызовите функцию <code>isPrime</code> и выведите результат.</p>

Входные данные

<p>Первая строка &mdash; целое число <code>N</code> &mdash; количество чисел (1 &le; N &le; 20).</p>
<p>Вторая строка &mdash; <code>N</code> целых чисел через пробел (числа положительные).</p>

Выходные данные

<p>Для каждого числа: <kbd>число: yes</kbd>, если число простое, или <kbd>число: no</kbd> в противном случае. Каждая запись с новой строки.</p>
