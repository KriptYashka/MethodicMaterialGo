Описание задачи

<p>В магазине &laquo;Четвёрочка&raquo; выбирают лучшего сотрудника месяца. Покупатели оставляют отзывы с именем сотрудника. Напишите три функции для подсчёта результатов:</p>

<ul>
<li><code>addVote(results map[string]int, name string)</code> &mdash; добавляет один голос за сотрудника. Если сотрудника ещё нет &mdash; функция добавляет его в словарь.</li>
<li><code>winner(results map[string]int) string</code> &mdash; возвращает имя сотрудника с наибольшим количеством голосов.</li>
<li><code>totalVotes(results map[string]int) int</code> &mdash; возвращает общее количество голосов.</li>
</ul>

<p>В программе вводится количество отзывов, затем имена сотрудников. Сохраните имена в срез, затем для каждого имени вызовите <code>addVote</code>. В конце используйте <code>winner</code> и <code>totalVotes</code> для вывода результата.</p>

Входные данные

<p>Первая строка &mdash; целое число <code>N</code> &mdash; количество отзывов (1 &le; N &le; 100).</p>
<p>Затем <code>N</code> строк, каждая с именем сотрудника (одно слово).</p>

Выходные данные

<p>Одна строка в формате: <kbd>Лучший сотрудник: X, голосов: Y из Z</kbd>, где X &mdash; имя победителя, Y &mdash; его голоса, Z &mdash; общее количество голосов.</p>
