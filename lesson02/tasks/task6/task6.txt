Описание задачи

<p>Магазин &laquo;Четвёрочка&raquo; проводит конкурс &laquo;Лучший покупатель месяца&raquo; среди владельцев бонусных карт. Вы — менеджер магазина. Вам нужно ввести данные участников и определить победителя &mdash; того, кто накопил больше всех баллов.</p>

<p>Сначала введите число <code>N</code> &mdash; количество участников. Затем введите <code>N</code> строк, каждая в формате &laquo;фамилия баллы&raquo;. Программа выведет всех участников и объявит победителя.</p>

<p>Подсказка: найдите максимум в цикле <strong>for range</strong> по map, сохраняйте имя лидера.</p>

Входные данные

<p>Сначала целое число <code>N</code> &mdash; количество участников. Затем <code>N</code> строк, каждая в формате &laquo;фамилия баллы&raquo; (баллы &mdash; целое число).</p>

Выходные данные

<p>Для каждого участника: <kbd>фамилия: баллы</kbd>. Затем строка <kbd>Победитель: фамилия</kbd>. Порядок вывода участников не имеет значения. При равенстве баллов победителем считается любой из лидеров.</p>
