Описание задачи

<p>Напишите функцию <code>sum(nums []int) int</code>, которая принимает срез целых чисел и возвращает их сумму.</p>

<p>Функция должна корректно работать с пустым срезом (возвращать 0).</p>

<p>В программе нужно сначала ввести количество чисел, затем сами числа, и вывести их сумму с помощью этой функции.</p>

Входные данные

<p>Первая строка &mdash; целое число <code>N</code> &mdash; количество чисел (0 &le; N &le; 100).</p>
<p>Вторая строка &mdash; <code>N</code> целых чисел через пробел (если N = 0, строка пустая).</p>

Выходные данные

<p>Одно целое число &mdash; сумма всех чисел.</p>
