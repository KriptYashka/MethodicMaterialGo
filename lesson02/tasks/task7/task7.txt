Описание задачи

<p>Сеть магазинов &laquo;Четвёрочка&raquo; состоит из нескольких филиалов. Вы — менеджер сети. Вам нужно собрать каталог товаров по всем филиалам. Данные храните в словаре словарей: <code>map[string]map[string]float64</code> &mdash; <em>филиал &rarr; товар &rarr; цена</em>.</p>

<p>Сначала введите число <code>N</code> &mdash; сколько филиалов. Для каждого филиала введите его название и число <code>M</code> &mdash; сколько в нём товаров, а затем <code>M</code> строк &laquo;товар цена&raquo;. Программа выведет каталог и определит самый дорогой товар среди всех филиалов.</p>

<p>Для обхода вложенного словаря используйте два цикла <strong>for range</strong>.</p>

Входные данные

<p>Сначала целое число <code>N</code> &mdash; количество филиалов. Для каждого филиала: строка с названием филиала и целым числом <code>M</code>, затем <code>M</code> строк в формате &laquo;товар цена&raquo; (цена &mdash; вещественное число).</p>

Выходные данные

<p>Для каждого товара: <kbd>филиал — товар: цена</kbd> (с двумя знаками после запятой). Затем строка <kbd>Самый дорогой: товар (филиал)</kbd>.</p>

<p>Порядок вывода товаров не имеет значения.</p>
