Введение

Продолжаем развивать программу управления магазином «Четвёрочка». Теперь мы умеем писать функции и работать со словарями (map).

Возьмите свою программу с прошлого занятия и модернизируйте её. В этом проекте вы вынесете повторяющуюся логику в функции и добавите новую возможность — скидки на товары.

Часть 1. Напишите три функции

calculateDiscountedPrice

Принимает цену и процент скидки, возвращает цену со скидкой.

Go 
func calculateDiscountedPrice(price, discountPercent float64) float64

printProductLine

Выводит строку каталога: номер, название, цена, скидка, цена со скидкой. Внутри себя вызывает calculateDiscountedPrice.

Go 
func printProductLine(number int, name string, price, discount float64)

Если скидка не установлена — выводите прочерк («—»).

findProduct

Ищет товар по названию в срезе names. Возвращает индекс или -1, если не найден.

Go 
func findProduct(names []string, product string) int

Часть 2. Добавьте словарь скидок

В main() объявите словарь:

Go 
discounts := make(map[string]float64)

Ключ — название товара, значение — процент скидки (например, 10.0 означает 10%).

Если на товар нет скидки — при обращении к discounts[name] вернётся 0.0. Передавайте 0 в printProductLine, и функция отобразит прочерк.

Часть 3. Новые пункты меню

Добавьте в меню три новых пункта (после прежних 9):

10. Установить скидку 11. Показать каталог со скидками 12. Очистить все скидки 13. Выход (или переместите Выход на 13)
В исходной программе пункт 9 — «Выход». Перенесите его на 13 или оставьте на 9, а новые пункты добавьте как 10, 11, 12. Выберите любой вариант.

Пункт 10. Установить скидку

  1. Запросить название товара.
  2. Через findProduct проверить, что товар существует. Если нет — вывести «Товар не найден.»
  3. Запросить процент скидки.
  4. Сохранить в discounts[name] = discount.

Пункт 11. Показать каталог со скидками

  1. Вывести заголовок таблицы: № Название Цена Скидка Цена со скидкой.
  2. Для каждого товара из names получить скидку из discounts.
  3. Вызвать printProductLine для вывода строки.

Пункт 12. Очистить все скидки

  1. Присвоить discounts = make(map[string]float64).
  2. Вывести «Все скидки очищены.»
