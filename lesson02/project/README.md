# Магазин "Четвёрочка" — Функции и скидки

Продолжаем развивать программу управления магазином. Теперь мы умеем писать **функции** и работать со **словарями (map)**.

Возьмите свою программу с прошлого занятия и модернизируйте её.

## Что нужно сделать

### Часть 1. Напишите три функции

```go
// Принимает цену и процент скидки, возвращает цену со скидкой
func calculateDiscountedPrice(price, discountPercent float64) float64

// Выводит строку каталога: номер, название, цена, скидка, цена со скидкой
func printProductLine(number int, name string, price, discount float64)

// Ищет товар по названию, возвращает его индекс или -1, если не найден
func findProduct(names []string, product string) int
```

Функция `printProductLine` должна использовать `calculateDiscountedPrice` внутри себя.

### Часть 2. Добавьте словарь скидок

В `main()` объявите `discounts := make(map[string]float64)`.
Ключ — название товара, значение — процент скидки (например, `10.0` — 10%).

### Часть 3. Новые пункты меню

Добавьте в меню три новых пункта (после原有的 9):

```
10. Установить скидку
11. Показать каталог со скидками
12. Очистить все скидки
```

**10. Установить скидку**
— Запросить название товара.
— Через `findProduct` проверить, что товар существует. Если нет — вывести "Товар не найден."
— Запросить процент скидки.
— Сохранить в `discounts`.

**11. Показать каталог со скидками**
— Вывести таблицу: `№  Название  Цена  Скидка  Цена со скидкой`
— Для каждого товара проверить, есть ли на него скидка в `discounts`.
— Использовать `printProductLine` для вывода.

**12. Очистить все скидки**
— Присвоить `discounts = make(map[string]float64)`.
— Вывести "Все скидки очищены."

## Критерии приёмки

- Все три функции написаны и используются
- Словарь `discounts` используется для хранения скидок
- Новые пункты меню (10–12) работают без ошибок
- Старые пункты меню (1–9) продолжают работать как раньше
- Код отформатирован (`go fmt`)
