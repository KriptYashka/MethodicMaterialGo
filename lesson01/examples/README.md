# Магазин "Четвёрочка" — управление товарами

Вы — менеджер продуктов в сети магазинов **"Четвёрочка"**.
Ваш отдел отвечает за небольшой ассортимент товаров.
Нужно написать программу, которая поможет управлять складом: добавлять новые позиции,
удалять устаревшие, анализировать цены и сортировать товары.

## Что нужно сделать

Напишите программу на Go, которая работает с двумя срезами:

- `names  []string`  — названия товаров
- `prices []float64` — цены товаров (в рублях)

Оба среза всегда синхронизированы: `names[i]` стоит `prices[i]`.

Программа показывает **меню** и выполняет выбранное действие:

1. **Добавить товар** — запросить название и цену, добавить в конец срезов
2. **Удалить товар** — запросить номер позиции и удалить элемент из обоих срезов
3. **Показать все товары** — вывести таблицу: № | Название | Цена
4. **Средняя стоимость** — посчитать и вывести среднюю цену
5. **Сортировка по возрастанию** — отсортировать товары от дешёвых к дорогим
6. **Сортировка по убыванию** — отсортировать товары от дорогих к дешёвым
7. **Самый дешёвый товар** — найти и вывести товар с минимальной ценой
8. **Самый дорогой товар** — найти и вывести товар с максимальной ценой
9. **Выход**

### Технические требования

- Вся логика — в функции `main()`, функции не используем (не пройдены)
- Сортировка — **своими руками** (например, пузырьковая), без `sort` и `slices`
- Для удаления — срез через `append(s[:i], s[i+1:]...)`
- Цикл меню — `for { ... }` с `break`
- Ввод — `fmt.Scan` или `fmt.Scanln`
- Вывод цен — с двумя знаками после запятой (`%.2f`)
- На старте в срезах уже лежат 3-5 товаров (хардкодом)
