Описание задачи

<p>Преподаватели и ученики мечтают, чтобы в Школе Программистов стало ещё больше новых спецкурсов. Давайте вызовем звездопад, чтобы каждый мог загадать это желание, увидев&nbsp;падающую звезду. Выведите в строку <strong>n</strong> звездочек, где <strong>n</strong> &ndash; количество людей, смотрящих на звездопад.</p>

Входные данные

<p>Вводится целое число <strong>n</strong> (1&lt;n&lt;100).</p>

Выходные данные

<p>Выводится <strong>n</strong> звездочек в строку без пробелов.</p>

Примечание: 

В Питоне:

Чтобы печатать звёздочки в одну строку используйте print("*", end="").
